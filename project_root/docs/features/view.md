# 갱신 조건 (Refresh Triggers)
ViewModel은 다음 상황에서 식물 목록을 새로고침(fetch) 해야 한다:
1. 새로운 식물을 등록(Create) 했을 때
2. 물/비료 주기 버튼 액션이 성공했을 때
3. 기존 식물 데이터가 삭제(Delete)되거나 수정(Update)되었을 때
4. 앱이 백그라운드에서 포그라운드로 전환될 때 00시가 지났는지 체크하여 업데이트
